# landingpage

> Human Relations' pre-launch landing page.

## Installation

``` bash
# install dependencies
$ make install

# build project and open in browser
$ make run

# build projects
$ make build
```

## Built with

- [Tailwind CSS](https://tailwindcss.com/)
- [TypeForm](https://typeform.com)
- [SendGrid](https://sendgrid.com/)
